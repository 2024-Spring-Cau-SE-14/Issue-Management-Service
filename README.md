# Issue-Management-Service

## Web-Server

### Build Executable Jar
```
cd web-server
gradlew build
```

### Run 
```
java -jar "build\libs\Web_Server-0.0.1-SNAPSHOT.jar"
```

### Website
```
http://localhost:8080
```
