# Issue-Management-Service

## Web-Server
```
cd web-server
```

### Set Database Connection
```
db-connection.bat
```

### Build Executable Jar
```
gradlew build
```

### Run 
```
java -jar "build\libs\Web_Server-0.0.1-SNAPSHOT.jar"
```

### Website Link  &nbsp; [Visit](http://localhost:8080/)
```
http://localhost:8080
```
